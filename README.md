## ラッシュアワー
ラッシュアワーというゲームです.

## 実行
```
$ git clone git@github.com:ohmuraken/rushHour.git
$ cd rushHour
$ javac rushHour/src/main/RushHour.java
$ java rushHour/src/main/RushHour
```
